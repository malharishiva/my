kiranreddyrebel Kodi repository for add-ons to stream Indian OTT Content

Project maintained by kiranreddyrebel



Add-ons

Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://kiranreddyrebel.github.io/ (Give it the name "kiranreddyrebel Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "kiranreddyrebel Repository", and install repository.kiranreddyrebel.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "kiranreddyrebel Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
